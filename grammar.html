How to set up (4 tabs in one Google Sheet)

1
sentences tab (required) — headers: cefr grammar_point sentence answer rule example distractors. Put ___ in the sentence where the gap is. distractors is optional.
2
verbs tab (for Beat the Clock) — headers: cefr base third_person past_simple past_participle ing_form.
3
adjectives tab (optional) — headers: cefr base comparative superlative adverb.
4
word_banks tab (optional) — headers: group words (words separated by |, e.g. in|on|at).
5
For each tab: File → Share → Publish to web → that tab → .csv, then paste the URLs below, Test, and Generate the student link.
📝 Sentences CSV URL (required)
🏃 Verbs CSV URL (for Beat the Clock)
🔢 Adjectives CSV URL (optional)
📚 Word banks CSV URL (optional)
